### 2026-03-12 16:02 — 자동 수집 불가 22% (22,044건) 해결 방안 분석

#### 1. Spend Base + US Supply Chain + KAIST EEIO (~11,356건) — 학술/경제모델

원본은 Exiobase, WIOD 등 산업연관분석 데이터베이스에서 나온 2차 가공 데이터.

| 방법 | 설명 | 난이도 |
|------|------|--------|
| 원본 DB 직접 수집 | Exiobase, WIOD는 무료 공개 → API/직접 다운로드 가능 | 중 |
| 변환 로직 개발 | raw 데이터를 배출계수 형태로 가공하는 파이프라인 필요 | 상 |
| KAIST EEIO | 한국 특화 → 연구팀에 데이터 제공 요청이 현실적 | 중 (외부 협의) |

#### 2. Laptop PCF (~6,109건) — 제조사별 제품탄소발자국

Dell, HP, Lenovo 등 제조사 홈페이지에 PDF로 공개. 형식이 전부 다름.

| 방법 | 설명 | 난이도 |
|------|------|--------|
| 제조사별 PDF 파서 | 주요 5개 제조사만 만들면 대부분 커버 | 상 |
| PAIA API 활용 | 업계 표준 도구(Product Attribute to Impact Algorithm) API로 일괄 수집 | 중 |

#### 3. KEITI EPD (~3,082건) — 환경성적표지

데이터 자체는 공개이나 로그인 + 동적 페이지라 단순 크롤링 불가.

| 방법 | 설명 | 난이도 |
|------|------|--------|
| 브라우저 자동화 | Playwright로 로그인 후 페이지 렌더링 → 수집 | 중 |
| KEITI API 요청 | 기관에 직접 API 제공 요청 | 중 (외부 협의) |
| 공공데이터포털 | 일부 EPD 데이터가 API로 이미 공개되어 있을 가능성 확인 필요 | 하 |

#### 4. Carbon Catalogue (~845건) — CDP 기업 배출

CDP(탄소정보공개프로젝트) 유료 구독 서비스.

| 방법 | 설명 | 난이도 |
|------|------|--------|
| CDP 라이선스 구매 | 구매 시 API 접근 가능 | 하 (비용 발생) |
| Climate TRACE 대체 | climatetrace.org에서 유사한 기업/시설 단위 배출 데이터를 API로 무료 공개 | 하 |

#### 5. US EPA EnergyStar (~322건) — 에너지 효율

실제로 공개 API 존재 (`https://data.energystar.gov/`). SODA API로 JSON/CSV 직접 다운로드 가능.

| 방법 | 설명 | 난이도 |
|------|------|--------|
| source_registry에 API URL 추가 | 바로 수집 가능 | **하** |

#### 6. Commuting & Travel + Events (~330건) — 통근/출장/행사

원본 출처가 대부분 DEFRA(영국)와 EPA(미국). 이미 자동 수집 대상인 DEFRA/EPA xlsx에 해당 카테고리가 포함되어 있음.

| 방법 | 설명 | 난이도 |
|------|------|--------|
| 추출 로직 보완 | DEFRA/EPA xlsx에서 통근/출장 시트를 빠뜨리지 않으면 자연스럽게 수집됨 | **하** |

### 2026-03-12 16:07 — 하/중 난이도 수집 불가 항목 코드 수정

**코드 개선 내역:**

1. **EnergyStar API 등록** (하, ~322건)
   - `source_registry/US.json`에 EnergyStar 소스 추가
   - SODA API CSV 엔드포인트 3개 (Furnace/Desktop/Display)
   - 상태: 수집 가능 (API 200 OK 확인)

2. **Climate TRACE API 등록** (중, Carbon Catalogue ~845건 대체)
   - `source_registry/INTL.json`에 ClimateTRACE 소스 추가
   - API 엔드포인트: `api.climatetrace.org/v6/country/emissions`
   - 국가/섹터별 CO2/CH4/N2O 배출량 무료 제공
   - 상태: 수집 가능 (API 200 OK, KOR 데이터 확인)

3. **KEITI EPD API URL 추가** (중, ~3,082건)
   - `source_registry/KR.json`에 epd.or.kr 공개 API URL 추가
   - 상태: 공공데이터포털(data.go.kr) 인증키 발급 필요 → 키 발급 후 즉시 수집 가능

4. **Commuting/Travel 추출 확인** (하, ~330건)
   - extractor에 commuting, business_travel, events 카테고리 키워드 이미 존재
   - DEFRA/EPA xlsx 수집 시 해당 시트에서 자동 추출됨

5. **Fetcher JSON API 지원** (중)
   - `fetch_auto`에 `data_format: "json"` 처리 추가
   - `fetch_api` 결과에 tables/text 변환 로직 추가 (extractor 호환)
   - INTL 소스 중 data_urls 없는 5개 기관 자동 스킵 유지

**수동 대비 분석:**
- 하 난이도 ~652건: **코드 수정 완료** (재수집 시 반영)
- 중 난이도 ~3,927건: Climate TRACE 완료, KEITI는 API 키 발급 대기
- 자동 수집 불가 비율: 22% → **~18%** (하+중 해결 시)
